# Networking Projects

This folder contains complete network implementations that combine multiple networking concepts into a single topology.

Unlike the individual labs, these projects simulate realistic network environments where multiple technologies work together.

## Goals

- Design practical enterprise-style network topologies
- Combine multiple networking concepts
- Improve troubleshooting skills
- Document the complete implementation process



## Current Projects

### Office Network using VLANs + RIP

This project simulates a small office environment where different departments communicate securely while remaining logically separated using VLANs.
