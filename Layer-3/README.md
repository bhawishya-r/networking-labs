# Layer 3 - Routing

## Overview

This directory contains my Cisco Packet Tracer labs focused on Layer 3 networking concepts.

Unlike Layer 2 switching, routers operate at the Network Layer (Layer 3) and are responsible for forwarding packets between different networks using IP addresses and routing tables.

Each lab is built from scratch to understand not only the configuration process but also how packets travel through a network and how routers make forwarding decisions.



## Objectives

- Understand Layer 3 communication
- Configure routers
- Build multi-router topologies
- Learn static and dynamic routing
- Understand routing tables
- Explore packet forwarding decisions
- Practice troubleshooting routing issues
