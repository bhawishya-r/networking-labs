# Networking Labs

A collection of networking labs built while learning Computer Networking.

These labs are created using Cisco Packet Tracer to practice networking concepts through hands-on implementation rather than theory alone.

## Technologies

- Cisco Packet Tracer
- Cisco IOS CLI
- IPv4
- Ethernet
- VLANs
- Static Routing
- DHCP
- DNS

## Repository Structure

- Layer 2
- Layer 3
- Projects

## Goal

To document my networking journey through practical labs, packet analysis, troubleshooting, and network design.

More labs will be added as I continue learning.
