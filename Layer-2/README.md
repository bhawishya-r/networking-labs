# Layer 2 - Data Link Layer Labs

This directory contains practical Cisco Packet Tracer labs focused on Layer 2 networking concepts.

The objective of these labs is to understand how switches forward frames, prevent loops, divide networks using VLANs, and build reliable local area networks.

## Topics Covered

- MAC Address Learning
- Collision Domains
- Broadcast Domains
- VLANs
- Access Ports
- Trunk Ports
- Spanning Tree Protocol (STP)

## Labs

| Lab | Description |
|-----|-------------|
| STP Lab | Preventing Layer 2 loops using Spanning Tree Protocol |
| VLAN Lab | Creating VLANs and assigning switch ports |
| Access Ports | Configuring end devices inside VLANs |
| Trunk Links | Carrying multiple VLANs between switches |

More labs will be added as I continue learning networking.
